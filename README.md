# Entendo o Desafio - Site de Clínica Médica

Este desafio tem como objetivo criar um site completo de uma clínica médica, aplicando todos os conceitos aprendidos no módulo de HTML.

O projeto deve abordar temas como:

- Formulários  
- Estruturação e formatação de texto  
- Inserção de mídias 
- Tabelas  
- Outros recursos estudados durante as aulas

---

## Instruções do Desafio

O site deve conter um menu de navegação com quatro páginas:

- Página Principal  
- Sobre a clínica  
- Horário de Atendimento  
- Contato  

Cada página deve seguir uma estrutura consistente, contendo:

- **menu de navegação**  
- **header** com imagem  
- **conteúdo principal**  
- **footer** com informações da clínica

O objetivo é desenvolver um site funcional, organizado e visualmente agradável, aplicando os conceitos aprendidos durante o módulo.
